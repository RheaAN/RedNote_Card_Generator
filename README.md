🚀 RedNote Pro - 智能小红书爆款图文生成器 (V16.14)
RedNote Pro 是一个纯前端、零依赖、开箱即用的单文件 (Single-File) AI 生产力工具。
它将大语言模型（LLM）的推理能力与可视化的卡片排版引擎完美结合，帮助创作者一键实现“选题灵感 -> 爆款文案 -> 数据可视化 -> 自动配图 -> 批量高清导出”的完整自动化工作流。
✨ 核心特性 (Features)
👁️ 灵感副驾 (Vision Copilot)
* 头脑风暴与选题搜索：不知道写什么？右侧副驾不仅能看图，还是你的专属策划大师！你可以直接与它对话：“帮我找一下最近小红书最火的搞钱副业话题”或“给我10个爆款选题灵感”，确定思路后再去左侧一键生成。
* 全局分屏聊天窗：右侧内置独立 AI 助手，支持物理分屏（Split Mode），随时对话，绝不遮挡主画布。
* 多模态视觉识别：完美支持上传本地图片，动态识别图片格式，无论是 Gemini 还是 Claude/GPT 都能精准“看”懂你的参考图并给出专业建议。
🧠 双核 AI 创作引擎 (Dual-Core AI Engine)
* 左侧自动化引擎：支持 Gemini 与 Claude/GPT（自定义中转接口）双通道。输入确定的选题后，一键输出爆款标题、带 Emoji 的排版正文、结构化图表数据及生图 Prompt。
* 自定义系统提示词 (System Prompt)：完全开放 AI 大脑控制权，支持高度定制化人设。修改后自动通过 LocalStorage 持久化保存，刷新不丢失。
* 智能 JSON 修复引擎：内置极客级词法分析器，自动拦截并修复 AI 输出的非法换行符、多余逗号等格式幻觉，极大提升自动化生成的成功率和稳定性。
🎨 骨灰级视觉排版 (Pro-Level Layout)
* 12 款预设主题：内置极简白、赛博朋克、麦肯锡、新中式等多种调性，一键全局切换。
* 自定义风格记忆：支持自由调节背景色、文字色、字体、行距，并可保存为“📂 我的预设”，随用随取。
* 智能封面排版：支持上/下/左/右四种图文布局。采用先进的 Flexbox 引擎，在左右排版时自动智能计算宽度比例，确保文字永远不会被挤压变形。
📊 丰富的可视化图表 (Data Visualization)
* 内置 10 种轻量级纯 CSS/DOM 图表组件，告别臃肿的图表库：金字塔、矩阵图、漏斗图、逻辑树 (Mindmap)、时间轴、动态循环图、九宫格、对比图、冰山模型、流程图。
* 图表文字及边框颜色随全局主题智能同步更新。
📂 本地数据导入与工程回滚 (Data Import & Rollback)
* Excel/CSV 自动生成卡片：支持点击左上角“📂 导入”按钮上传 .csv 文件。你不仅可以一键恢复之前的工程状态，还可以直接使用 Excel 批量修改文案、参数和图表数据，上传后系统将瞬间自动渲染生成整套图文卡片，实现工业化批量量产！
📦 工业级打包导出 (Industrial Export)
* 无痕下载配图：配图生成采用 fetch Blob + FileReader 转换为 Base64 本地化存储，彻底根除了 html2canvas 跨域污染画布 (CORS Tainted) 导致的白屏 Bug。
* 一键 ZIP 终极打包：点击右上角“📦 打包”按钮，你将获得一个包含以下完整资产的压缩包：
   1. RedNote_1.png, RedNote_2.png...：完美切片的高清卡片成品图，直接发小红书。
   2. project_data.csv：工程状态备份文件。可用 Excel 打开并修改，也可用于后续回滚。
   3. chat_history.txt：你和右侧灵感副驾的完整聊天记录，留存所有的灵感火花。
   4. readme.txt：包含本次创作的完整纯文本内容、标题、作者和图表核心数据。
🛠️ 技术栈 (Tech Stack)
本项目极致轻量，无需 npm install，无需任何后端或构建工具，双击单个 HTML 文件即可运行全部功能：
* 核心骨架：HTML5 + 原生 JavaScript (Vanilla JS)
* 样式框架：Tailwind CSS (CDN)
* 核心依赖库 (均通过 CDN 引入)：
   * html2canvas：实现 DOM 到高清图片的无损转换。
   * JSZip & FileSaver.js：实现本地大文件的打包与下载闭环。
   * marked.js：解析 AI 助手的 Markdown 对话。
 

🚀 快速上手 (Getting Started)
1. 运行项目
将代码 clone 到本地，直接在现代浏览器（推荐 Chrome 或 Edge）中双击打开 RedNote_Pro_V16.14_Stable.html 即可运行。
2. 配置 AI API Key
* 本工具完全运行在本地，不自带大模型 API 额度，需填入你自己的 API Key。
* 右侧灵感副驾：打开右侧面板，填入 Gemini Key 或 中转/自定义 Base URL 与 Key，用于前期的头脑风暴和识图。
* 左侧内容引擎：在“🤖 AI 智能生成”区块，填入同样的 Key，用于后续的一键自动化生产。
* 注：所有的 API Key 和接口地址仅保存在你当前浏览器的 LocalStorage 中，绝对安全，不会上传至任何第三方服务器。
  
🧪 核心使用与测试指南 (Testing Guide)
如果你是第一次使用，请按照以下标准工作流进行体验：
1. 第一阶段：找灵感 (右侧副驾)
   * 点击右上角魔法棒图标 ✨ 打开灵感副驾，切换为“分屏模式”。
   * 输入：“帮我找3个适合在春天发布的小红书治愈系选题”。
   * 如果你有参考图，点击 + 号上传图片，让它帮你分析图中的卖点或文案套路。
2. 第二阶段：调校人设与一键生成 (左侧引擎)
   * 在左侧找到“系统人设”文本框，你可以任意修改（如：要求语气幽默、多用叹号），修改后自动保存。
   * 将右侧讨论好的选题输入到“用户选题”框中。
   * 点击“✨ 一键生成图文”，观察系统如何自动解析并填充标题、正文、配图建议和图表数据。
3. 第三阶段：视觉包装
   * 封面与排版：上传一张好看的封面图，尝试切换“左图右文”或“上图下文”。
   * 全局主题：点击“🎨 风格模板”中的预设（如“麦肯锡”），测试图表颜色是否完美同步。
   * 保存预设：调整好自己喜欢的字号、行距、颜色后，点击“💾 保存配置”，它会永久出现在下方的“📂 我的预设”中。
4. 第四阶段：终极导出
   * 点击左侧的“🎲 生成并下载配图”，等待自动抓取版权免费配图。
   * 点击页面顶部导航栏的黑底“📦 打包”按钮。
   * 解压下载的 ZIP 文件，检查高清卡片图片和附带的 CSV 工程备份！
5. 第五阶段：Excel 批量修改与回滚 (隐藏黑科技)
   * 刷新网页清空所有内容。
   * 点击左上角“📂 导入”按钮，选择刚才下载打包里的 project_data.csv，见证整个页面内容和样式瞬间还原！
   * 你甚至可以用 Excel 打开这个 CSV 文件，直接把里面的文案替换成新的内容，上传后即可全自动渲染成新卡片。
